# Codex Switch

Desktop app for managing provider configs for Claude Code, Codex, Gemini CLI, OpenCode, and OpenClaw.

## Development

```bash
corepack pnpm install
corepack pnpm dev
corepack pnpm test:unit
cargo test --manifest-path src-tauri/Cargo.toml
```

## Build

```bash
corepack pnpm build
```
