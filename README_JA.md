# Codex Switch

Claude Code、Codex、Gemini CLI、OpenCode、OpenClaw のプロバイダー設定をまとめて管理するデスクトップアプリです。

## 開発

```bash
corepack pnpm install
corepack pnpm dev
corepack pnpm test:unit
cargo test --manifest-path src-tauri/Cargo.toml
```

## ビルド

```bash
corepack pnpm build
```
